# Pixel Art

A pixel art editor for mytekDev, based on [Piskel](https://github.com/piskelapp/piskel).

## Credits

This application is built on top of [Piskel](https://www.piskelapp.com/), an open-source pixel art and sprite editor created by Julian Descottes ([@juliandescottes](https://github.com/juliandescottes)).

- **Original project**: [github.com/piskelapp/piskel](https://github.com/piskelapp/piskel)
- **License**: Apache License 2.0

## Changes from upstream

- Integrated with the mytekDev platform (project saving, sharing, asset library)
- Custom styling to match the platform theme
